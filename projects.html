Tish - CLI Shell

Systems Programming

Overview

Tish is a command-line shell built from scratch in C/C++. This project demonstrates deep understanding of low-level systems programming, process management, and Unix shell design patterns.

Key Features

  • Command Parsing: Robust parsing of user commands with support for arguments and options
  • Process Management: Fork and execute system processes with proper signal handling
  • Piping: Support for command piping and output redirection
  • Built-in Commands: Implementation of essential shell commands like cd, exit, history
  • Error Handling: Comprehensive error handling and user feedback
  • Environment Variables: Management and expansion of shell variables
